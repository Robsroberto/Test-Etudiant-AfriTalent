# AfriTalent

**Nom et classe :** Test Etudiant - L1GL - Groupe ISI

## Description

Plateforme de mise en relation entre entreprises et freelances africains, realisee
dans le cadre du module HTML/CSS/JS - Semestre 2.

## Technologies utilisees

- HTML5 semantique
- CSS3 (variables, Grid, Flexbox, Bento Grid, media queries)
- JavaScript vanilla (IntersectionObserver, localStorage)
- Bootstrap 5

## Fonctionnalites principales

- 5 pages liees par une navigation commune (Accueil, Freelances, Tarifs, A propos, Contact)
- Mode sombre persistant entre les pages (localStorage)
- Navbar dynamique au scroll
- Filtrage dynamique des freelances par categorie
- Compteurs animes et animations fade-in au scroll
- Formulaire de contact avec validation JavaScript complete
